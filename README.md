## 📖 About

CyberPass Analyzer is a responsive web application that helps users create stronger and more secure passwords. It evaluates password strength in real time, calculates password entropy, generates secure random passwords, and provides an intuitive user experience through interactive visual feedback.

This project was developed to strengthen my understanding of front-end web development concepts, including DOM manipulation, event handling, JavaScript logic, and responsive UI design.

## 🧠 How It Works

The application evaluates passwords using five essential security rules:

- ✅ Minimum 8 characters
- ✅ Uppercase letter
- ✅ Lowercase letter
- ✅ Number
- ✅ Special character

Each satisfied condition increases the strength score.

Based on the score, the password is classified as:

- 🔴 Weak
- 🟠 Medium
- 🟢 Strong

The application also calculates password entropy based on the detected character set to provide a measure of password complexity.
